## 参照元として想定している一次資料と一般基準

外部リンクは含めない。
必要な場合は、組織内で参照可能な形で各自が入手する前提。

- Usability Heuristics（10原則）
  - ユーザビリティ監査の根拠として使われる代表的な枠組み
- Human-Centered Design（人間中心設計）
  - ユーザー理解、反復、全体体験、チーム協働などの原則
- Web Content Accessibility Guidelines（WCAG）
  - コントラスト、キーボード操作、エラー伝達などの基準
- WAI-ARIA Authoring Practices
  - WebのUI部品のセマンティクスと操作の推奨
- Apple Human Interface Guidelines
  - iOSにおける操作、レイアウト、タップ領域、文言の考え方
- Android Design / Accessibility Guidance
  - Androidにおけるタッチターゲット、ナビゲーション、アクセシビリティの考え方
- Material Design
  - コンポーネント、タイプスケール、色システムなどの考え方
- 日本語圏のデザインシステム公開例（企業のデザインシステムやチェックリスト）
  - UI一貫性、文言、アクセシビリティ、運用の参考として
